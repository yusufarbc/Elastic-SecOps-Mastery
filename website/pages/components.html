Stack Components

Detailed reference for Elasticsearch, Logstash, and Kibana

Elasticsearch

Configuration

  • Port: 9200 (Localhost Only)
  • Security: TLS Enabled, RBAC
  • Heap Size: 4GB (default)
  • Config Path: /etc/elasticsearch/elasticsearch.yml

Features

  • Index Lifecycle Management (90 days)
  • Data Streams support
  • Self-signed Certificate Authority

Useful Commands

# Check Health
curl -s --cacert /etc/elasticsearch/certs/ca.crt https://localhost:9200/_cluster/health?pretty

# List Indices
curl -s --cacert /etc/elasticsearch/certs/ca.crt -u elastic:PASSWORD https://localhost:9200/_cat/indices?v

Logstash

Pipeline Ports

WEF 5045/tcp
Syslog 5514, 5515
Kaspersky 5516

Processing

  • ECS Normalization
  • GeoIP Enrichment
  • Date Parsing & Grok
  • Persistent Queues
