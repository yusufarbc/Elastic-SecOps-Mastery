Troubleshooting

Diagnose and resolve common installation and operational issues.

Elasticsearch Issues

Service Won't Start

Common causes: Memory, vm.max_map_count, or Permissions.

# Check memory
free -h

# Check logs
sudo journalctl -u elasticsearch -n 50 --no-pager
Certificate Errors

Verify permissions on /etc/elasticsearch/certs

ls -la /etc/elasticsearch/certs/
sudo chown root:elasticsearch /etc/elasticsearch/certs/*

Kibana Issues

Enrollment Token Expired

Tokens are valid for 30 minutes. Generate a new one:

sudo /usr/share/elasticsearch/bin/elasticsearch-create-enrollment-token -s kibana
Kibana Server Not Ready

Kibana cannot connect to Elasticsearch. Check elasticsearch.hosts in configuration.

sudo journalctl -u kibana -n 50 --no-pager

Logstash Issues

Pipeline Won't Start

Validate your configuration syntax:

sudo /usr/share/logstash/bin/logstash --path.settings /etc/logstash -t
Authentication Failure

Check the keystore for ES_PW:

sudo /usr/share/logstash/bin/logstash-keystore --path.settings /etc/logstash list
